# 🎨 拼豆图纸转换器 - 使用说明

## 快速启动

### 方法一：直接用浏览器打开
1. 双击 `index.html` 文件
2. 在浏览器中使用（部分功能可能受跨域限制）

### 方法二：本地服务器（推荐）

**macOS / Linux:**
```bash
cd perler-bead-converter
python3 -m http.server 8080
```

**Windows:**
```bash
cd perler-bead-converter
python -m http.server 8080
```

然后在浏览器访问：`http://localhost:8080`

## 功能说明

- 📷 上传任意图片
- 🎨 自动转换为拼豆图纸
- 📊 显示每种颜色的使用数量
- 📐 支持多种拼豆尺寸（2.6mm/5mm/10mm）
- 🎯 可选网格/圆点输出样式
- 💾 一键下载图纸

## 颜色库

内置 36 种标准拼豆颜色，涵盖常用色系。
